API Sudskog registra

Ovaj API omogućuje automatsko dohvaćanje podataka o tvrtkama iz Sudskog registra (sudreg-data.gov.hr) prilikom dodavanja novih klijenata ili dobavljača.

  • API URL: https://sudreg-data.gov.hr/api/javni
  • Testno okruženje: https://sudreg-data-test.gov.hr/api/javni
  • Autentifikacija: OAuth2 Client Credentials (client_id i client_secret)
  • Dokumentacija: sudreg-data.gov.hr/api/javni/dokumentacija/open_api
